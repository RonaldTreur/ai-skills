# Agent Methodology Adaptation

This folder tracks how external agent-methodology sources influence this ai-skills repository.

It exists for three reasons:

1. Compare external sources by development discipline instead of copying skill packs wholesale.
2. Keep attribution clear enough to credit people and explain what changed.
3. Make future re-reviews possible when external projects evolve.

Use the `external-skill-adaptation` skill when adding to this folder.

## Structure

- `DISCIPLINES.md`: the discipline map for the user-facing project-development routine.
- `ADAPTATION_LOG.md`: provenance ledger for adopted, adapted, inspired, rejected, deferred, and superseded ideas.
- `sources/`: source inventories for external projects and instruction sets.
- `disciplines/`: one review document per discipline.
